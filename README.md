# PhoneBookLab

**Лабораторна робота №5**  
Тема: Розробка програм з файловими змінними. Робота з файлами  

**Автор:** Устич Максим, група alk43  
**Рік:** 2025  

## Опис
Програма створює телефонний довідник, що містить:  
- Прізвище  
- Ім'я  
- По-батькові  
- Номер телефону  

Дані вводяться з консолі, зберігаються у файл `phonebook.txt`, а потім читаються та виводяться на екран.  

## Як запустити
1. Відкрити проект у Visual Studio 2022.  
2. Скомпілювати проект.  
3. Запустити програму.  
4. Ввести дані абонентів.  
5. Переглянути результат у консолі та файл `phonebook.txt`.
